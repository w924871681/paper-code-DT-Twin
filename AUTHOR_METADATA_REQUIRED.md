# Author metadata required before a non-anonymous submission

No author identity, affiliation, funding, DOI, or contact value has been
inferred. Please provide and approve the following before replacing anonymous
submission metadata:

- author order;
- full English names;
- affiliations;
- corresponding author;
- emails;
- ORCID identifiers;
- CRediT roles;
- funding agency;
- grant number;
- conflict-statement confirmation;
- target journal anonymity policy.

For a double-blind submission, retain the anonymous manuscript and prepare a
separate non-anonymous final template. For a non-double-blind submission,
replace the placeholders only after the values above have been confirmed.
